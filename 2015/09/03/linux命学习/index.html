linux命学习

  • 技术

1 查找pid

  • ps aux | grep sshd
    • ps aux 显示系统所以进程
    • | 管道重定向，将管道命令之前的输出，重定向到管道命令后的命令中，作为该命令的输入。
      grep sshd过滤剩下与sshd相关的进程

2 services命令

命令:service name command：service httpd reload

  • 使用command方式控制名字为name的服务，上面是将http重新reload
  • service --status-all，列出所有系统服务的当前状态
  • command 命令如下表：
    | 命令 | 命令描述 |
    | ——– | —–: |
    | start | 启动指定的服务 |
    | stop | 通过感觉的方式停止指定服务 |
    | restart | 重新启动指定的服务 |
    | reload | 重新装载指定服务的配置文件 |
    | status | 显示指定服务的状态｜
  • service --status-all，列出所有

3 杀死进程

  • kill pid，杀死指定进程
  • kill -9 pid，强制杀死指定进程，信心可以突然报表
  • killall process name，如：killall httpd，表示结束某个名字的所有进程，同样名字可以有多个进程

4 ubuntu mysql utf8编码的设置：

到/etc/my.cnf或者是/etc/mysql/my.cnf里面修改client，mysqld,mysql字段，**网上很多人的路径是/etc/my.cnf,我的是在/etc/mysql/my.cnf**，怎么修改百度去

###5 service mysql start失败：
glinux@ubuntu:~$ sudo service mysql start
start: Job failed to start

问题出现的原因：我将`/etc/mysql/my.cnf`里面的[mysql]设置为`default-character-set=utf8`了，网络资料说：如果mysql版本5.5后设置mysqld字段设置如下：

character-set-server=utf8
