1、类const int，const static int，static int，int数据成员：

const int

数据成员变量不可以在类外定义，需要使用构造函数直接初始化而非赋值初始化，即只可以在类外部定义初始化
例如直接初始化：aClass(int num)：initConstInt(num)

const static int

数据成员变量可以在 类内、类外 定义初始化

static int

数据成员变量只可以在 类外部 定义初始化

int

数据成员只能在类外 定义

2、#宏

  • #ifdef _DEBUG
    如例子，在VS C++项目里面常使用到#ifdef _DEBUG告诉编译器：如果现在是debug调试状态，会编译DEBUG_CONFIG_TEST(x,y);否则执行的是RELEASE_CONFIG_TEST(x,y)，这样一个直接的用处是：根据不同需求进行一些特殊处理，我们可以DEBUG_CONFIG_TEST,RELEASE_CONFIG_TEST
    写两份不同的代码，调试的时候使用debug的代码，发行版本使用的是release的代码
  • #define
    define可以定义多行，但是需要行尾加入’\’，要牢牢记住宏原理，详情参考C/C++之define用法小结,如代码例子
  • 使用define的一些好处？
    • 函数调用的开销
    • 类型无关，如果一个int，double，float比较大小，可能需要重写三个函数，宏只要一个。
    • 决解一些函数无法完成的任务,比如有些参数类型必能传递给函数（设计为单例的类）
